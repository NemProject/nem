How to set-up the WebServer hosting the NEM-Code
================================================

NEM will be distributed via Java WebStart.
To enable this a few initial set-up tasks for the Web Server hosting the code have to be done.

1) Follow the directions documented for WebStart => http://docs.oracle.com/javase/7/docs/technotes/guides/javaws/developersguide/setup.html

2) Place the "nem-client.jnlp" on the web server

3) Place the HTML-pages and all subdirectories from webstart/html onto the server

4) Direct your browser to the server
The web page for starting NEM should be shown.

Have fun!
